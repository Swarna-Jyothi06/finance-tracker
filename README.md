# Personal Finance Tracker

## Overview
The Personal Finance Tracker is a web application developed using Spring Boot and HTML/CSS to help users manage and track their daily financial transactions.

Users can record expenses, categorize them, and view their financial activities in an organized manner.

---

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- Maven

---

## Features

### Module 1 – User Interface
- Navigation bar with Dashboard, Transactions, Budget, and Profile
- Clean and responsive design
- Transaction form interface

### Module 2 – Transaction Management
- Add financial transactions
- Select category
- Choose account type
- Add transaction date
- Save transaction data
